# Skybird Chrome Extension

Quick-entry popup for the Skybird travel agency app.

## Features
- Login with your Skybird (Supabase) email + password
- Create ticket (passenger, supplier, buyer, booking/travel dates, cost & sale)
- Record payment (in/out, cash/bank, customer/supplier/sub-agent)
- Recent tickets shown at the bottom

## Install
1. Download `skybird-extension.zip` and unzip it
2. Open `chrome://extensions`
3. Enable **Developer mode** (top-right)
4. Click **Load unpacked** and select the unzipped `extension/` folder
5. Pin the extension and click the airplane icon
